# mango plate clone

망고플레이트 앱 클론 프로젝트

지역마다의 맛집을 추천해주고 나만의 맛집을 설정할 수 있는 어플입니다.

식당마다 리뷰를 작성할 수 있으며, 해당 리뷰의 좋아요 수 등 점수를 매겨 오늘의 리뷰를 선정합니다.

* 기간: 2022/05/20 ~ 2022/06/03
* 팀구성: Server 개발자 2명, ios 개발자 2명
